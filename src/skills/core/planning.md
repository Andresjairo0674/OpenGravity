# Skill: Writing Plans (Redacción de Planes)

Esta habilidad se activa una vez que el diseño ha sido aprobado mediante la fase de Brainstorming.

## Objetivo
Crear un plan de implementación detallado, modular y verificable.

## Instrucciones para el Agente
1. Describe el objetivo final de forma clara.
2. Divide el trabajo en tareas pequeñas (máximo 5 min cada una).
3. Especifica las rutas de los archivos que serán creados o modificados.
4. Incluye pasos de verificación para cada tarea (ej: comandos de consola, logs esperados).
5. Guarda el plan en un artefacto visible para el usuario.
